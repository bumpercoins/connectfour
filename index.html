Bot Settings:

The number Of MCTS(Monte Carlo Tree Search) Simulations esentially controls the Bot strength. The minimum number is 1 and there is no maximum number. I set the default at 1000 because that's the setting that the Bot used when it "learned" to play via the AlphaZero self-play training process.

A nice property of MCTS is that the time it takes to execute roughly scales linearly with the number of MCTS Simulations. You can consider this analogous to the depth paramter in traditional alpha beta minimax tree search in that it controlls how "deep" you explore. However with minimax tree search, the time it takes scales exponentially with the depth parameter.
